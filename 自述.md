# 玉衡桌面環境

# 介紹
玉衡桌面環境是一個正在開發中的窗口管理器，注重功能、效率以及外觀。此外，它不會有煩人的問題，比如分數比縮放下模糊的 XWayland 窗口。

# 狀態
目前玉衡處於極早期的開發階段——甚至不曾成型。這意味着眼下我們不鼓勵一切形式的應用。
